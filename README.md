uso de Git y Github.

## ⚙️ Tecnologías Utilizadas
* Java JDK 17.0.13
* Maven 3.9.9
* Git 2.45.1

## 🛠️ Instalación y Uso
Para instalar y ejecutar el proyecto, sigue los siguientes pasos:

### Paso 1. Copiar el repositorio

```shell
git clone https://github.com/felistron/mx-uaemex-fi-25b-hello-world.git
cd mx-uaemex-fi-25b-hello-world
```

### Paso 2. Compilar y ejecutar el proyecto

Compilar el proyecto:
```shell
javac src/main/java/mx/uaemex/fi/Main.java -d target/classes
```

Ejecutar el proyecto:
```shell
java -classpath target/classes mx.uaemex.fi.Main
```
